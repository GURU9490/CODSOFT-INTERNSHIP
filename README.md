## 📄 Task 1 — Personal Portfolio

> **Objective:** Create a personal portfolio using HTML and CSS showcasing your name, skills, projects, resume, and contact information.

### 🔍 What's Included

- **Header / Hero** — Name, role tagline, profile photo (embedded), and CTA buttons
- **About Section** — Short bio with experience stats and a quote block
- **Skills Section** — Animated progress bars for each technology, scroll-triggered
- **Projects Section** — 4 project cards with descriptions and tech tags
- **Education Timeline** — Degree, certification, and school with year markers
- **Resume Section** — Prominent download button for PDF resume
- **Contact Section** — Contact info cards (email, phone, LinkedIn, GitHub) + message form

- CSS custom properties (variables) for consistent theming

## 🧮 Task 3 — Calculator

> **Objective:** Build a functional calculator using CSS, HTML, and JavaScript with an interactive interface supporting addition, subtraction, multiplication, and division using event listeners, operators, and loops.

### 🔍 What's Included

- **Display Panel** — Two-row display: running expression + current value
- **Full Arithmetic** — Addition `+`, Subtraction `−`, Multiplication `×`, Division `÷`
- **Extra Functions** — `AC` (all clear), `+/-` (toggle sign), `%` (percentage)
- **Chained Operations** — Supports continuous calculations without pressing equals
- **Keyboard Support** — Full keyboard input for digits, operators, Enter, Backspace, Escape
- **Ripple Animation** — Material-style ripple effect on every button press
- **3D Button Press** — Buttons physically push down on click using CSS box-shadow
- **Error Handling** — Displays "Error" cleanly on division by zero

### 💡 Key Concepts Practised

- CSS Grid for button layout alignment
- JavaScript event listeners (`click`, `keydown`)
- Conditional logic (if-else) and arithmetic operators
- State management for multi-step calculations
- CSS `@keyframes` animation and `box-shadow` for 3D effects

### ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0` – `9` | Input digits |
| `+` `-` `*` `/` | Set operator |
| `.` | Decimal point |
| `Enter` / `=` | Calculate result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all (AC) |
